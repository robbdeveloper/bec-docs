# `[bec_version]`

Restituisce il **numero di versione** del plugin come testo semplice — utile quando invii screenshot al supporto.

---

## Dove posizionarlo

Appendice privacy, colonna footer nascosta, pagine checklist manutenzione — di norma non critico per i visitatori.

---

## Attributi

_Nessuno._

---

## Esempi

```
Versione plugin: [bec_version]
```
