# `[bec_version]`

Outputs the plugin **version number** as plain text—useful when sending screenshots to support.

---

## Where to put it

Privacy policy appendix, hidden footer column, maintenance checklist pages—not normally visitor-critical.

---

## Attributes

_None._

---

## Examples

```
Plugin version: [bec_version]
```
